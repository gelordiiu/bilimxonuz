## Qidiruv algoritmlari: Linear va Binary Search

Linear Search — barcha elementlarni tekshiradi, O(n). Binary Search — tartiblangan ro'yxatda o'rtadan qidiradi, O(log n). Binary Search faqat saralangan massivlarda ishlaydi. Recursion — o'z-o'zini chaqiruvchi funksiya.

## Kod Namunasi
```
def binary_search(arr, x):
    lo, hi = 0, len(arr)-1
    while lo <= hi:
        mid = (lo+hi)//2
        if arr[mid] == x: return mid
        elif arr[mid] < x: lo = mid+1
        else: hi = mid-1
    return -1

sorted_arr = [1,3,5,7,9,11]
print(binary_search(sorted_arr, 7))
```
