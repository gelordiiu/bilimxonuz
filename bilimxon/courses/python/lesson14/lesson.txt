## Singleton va Factory Pattern

Singleton — sinfning faqat bitta nusxasi bo'lishini ta'minlaydi (ma'lumotlar bazasi ulanishi, logger). Factory — ob'yekt yaratishni markazlashtiradi, konkret sinfni yashiradi. Kreativ naqshlar kategoriyasiga kiradi.

## Kod Namunasi
```
class Singleton:
    _instance = None
    def __new__(cls):
        if not cls._instance:
            cls._instance = super().__new__(cls)
        return cls._instance

a = Singleton()
b = Singleton()
print(a is b)  # True
```
